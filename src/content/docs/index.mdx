## Key Features

<CardGrid stagger>
	<Card title="Secure Isolation" icon="lock">
		Agents execute within isolated Docker containers with strict resource limits and read-only filesystems.
	</Card>
	<Card title="Human-in-the-Loop" icon="approve-check">
		Every on-chain transaction requires explicit user approval via a secure UI overlay.
	</Card>
	<Card title="Wallet Connect" icon="puzzle">
		Connect any mobile or browser wallet via WalletConnect, or use local Anvil nodes for testing.
	</Card>
	<Card title="AI Proxy" icon="rocket">
		Built-in proxies for OpenAI and Anthropic APIs allow agents to use LLMs without exposing your API keys to the agent code.
	</Card>
</CardGrid>
