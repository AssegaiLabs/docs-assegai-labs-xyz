## About

The Assegai AI Sandbox runs AI agents in Docker containers with spending controls and transaction approval for blockchain operations.

### Features

- **Docker Isolation**: Agents run in containers with resource limits
- **Transaction Approval**: All transactions require explicit user approval
- **Spending Controls**: Per-transaction, daily, weekly, and monthly limits
- **Token Allowances**: Configure allowances per token per chain
- **AI API Access**: Proxied OpenAI and Anthropic API access with cost tracking
- **Wallet Support**: WalletConnect and local development wallet modes

### Current Status

MVP in development. Core features are functional for early testing and co-development.
