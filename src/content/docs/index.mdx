## What is Assegai AI Sandbox?

The Assegai AI Sandbox is an cross-platform desktop application (macOS, Windows, Linux) that allows users to run sandboxed AI agents for on-chain blockchain activity safely.

## Key Capabilities

- **Sandboxed Execution**: Run AI agents in isolated environments
- **Approval Limits**: Configure approval limits per token
- **Spending Limits**: Set spending limits per token
- **Transaction Approval**: Optional user approval workflow via desktop notifications before executing transactions

Currently in MVP development. This documentation supports early testing and co-development efforts.
